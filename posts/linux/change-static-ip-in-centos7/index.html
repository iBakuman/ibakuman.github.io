在 CentOS 7 中配置静态 IP

2023/05/26 15:26:02
阅读时长: 1 分钟

编辑 /etc/sysconfig/network-scripts/ifcfg-ens30 文件，其中 ens30 是网卡的名称，改为自己的网卡名称。

1

vi /etc/sysconfig/network-scripts/ifcfg-ens30

将以下几行添加到该文件中

TYPE="Ethernet"
PROXY_METHOD="none"
BROWSER_ONLY="no"
BOOTPROTO="static"
DEFROUTE="yes"
IPADDR=<desired IP address>
NETMASK=<desired Subnet Mask>
GATEWAY=<desired Default Gateway>
DNS1=<desired DNS server>
NAME="ens32"
UUID="ab6ebc7c-0ea6-496f-8961-71ba199f3dcc"
DEVICE="ens32"
ONBOOT="yes"

请注意，将 <desired IP address>、<desired Subnet Mask>、<desired Default Gateway> 和 <desired DNS server> 替换为您要使用的实际IP信息。例如，如果您要使用IP地址为 192.168.0.100、子网掩码为 255.255.255.0、默认网关为 192.168.0.1，则 ifcfg-eth0 文件应如下所示：

GATEWAY=192.168.22.2
DNS1=8.8.8.8
NAME="ens32"
UUID="ab6ebc7c-0ea6-496f-8961-71ba199f3dcc"
DEVICE="ens32"
ONBOOT="yes"

重启网络服务以使更改生效

1

systemctl restart network
